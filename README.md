## 📦 What nthn is

nthn is a framework for keeping shared state in step across distributed systems. It helps teams build apps that need the same data on many machines at once.

This project is aimed at end users who want to run a Windows build from a release page. It gives you a simple path from download to first run.

## 🔧 How nthn fits your setup

nthn is built around state management in distributed systems. In simple terms, it helps software keep data aligned across several nodes.

You may see terms like:

- **Consensus**: how nodes agree on the same value
- **Byzantine fault tolerance**: how the system handles bad or broken nodes
- **Peer-to-peer**: how nodes talk without one central server
- **Hashgraph**: a way to share data and reach agreement
- **State abstraction**: a clean layer for working with shared state

You do not need to learn these terms to run the app, but they help explain what nthn is for.
